# XDU 自动连网脚本

> 西安电子科技大学校园网自动拨号工具（Windows）

## 简介

一键自动连接 XDU 校园网。支持自动重试、错误检测、日志记录，账号密码独立存储，可安全上传 GitHub。

## 文件说明

| 文件 | 说明 |
|------|------|
| `link_XDU.bat` | 主拨号脚本（双击运行） |
| `config.bat` | **你的账号密码配置**（.gitignore，不上传） |
| `config.bat.example` | 配置模板（上传，供参考） |
| `.gitignore` | Git 忽略规则 |
| `link_XDU.log` | 运行日志（自动生成） |

## 首次使用

1. 将 `config.bat.example` **复制**一份为 `config.bat`
2. 用记事本打开 `config.bat`，填入你的校园网账号和密码

```bat
set "ACCOUNT=你的学号"
set "PASSWORD=你的密码"
```

3. 双击 `link_XDU.bat` 即可自动连网

## 功能特性

- ✅ **自动重连** — 连接失败自动重试（最多 10 次，每次间隔 3 秒）
- ✅ **智能判断** — 已连接则跳过拨号，避免重复
- ✅ **错误分类** — 密码错误（691）直接退出，不浪费重试
- ✅ **双重确认** — 拨号后二次检测连接状态，防止误判
- ✅ **日志记录** — 每次运行结果写入 `link_XDU.log`
- ✅ **密码安全** — 账号密码独立存储在 `config.bat`，不会被 Git 追踪

## 常见问题

**Q: 提示「找不到 config.bat」？**
将 `config.bat.example` 复制为 `config.bat` 并填入账号密码。

**Q: 一直提示错误 691？**
检查 `config.bat` 中的账号密码是否正确，或校园网账户是否欠费。

**Q: 连接名称不是 XDU？**
用 `rasdial /?` 查看已有连接名称，修改 `link_XDU.bat` 中的 `CONN_NAME` 变量。
